Three full-stack builds, most recent first. Each one taught me something different — about auth, real-time data, or just shipping something that works end to end.

Cognica Quiz Application

October 2025

A topic-based quiz platform — the kind of project where the real challenge wasn't the quiz logic, it was getting auth and session handling solid.

  • Full-stack quiz web app with register, log in, and log out flows, with user data securely stored in MongoDB.
  • Presents technology-based quiz questions across multiple topics, gives real-time feedback, and tracks user performance.

DeepSeek Clone

June 2025

My deepest dive into AI-product UX so far — wiring up streaming responses, formatting, and auth in one cohesive chat experience.

  • Full-stack AI-powered chat application inspired by DeepSeek, built with Next.js and Tailwind CSS.
  • Used the OpenAI API to power intelligent, real-time AI responses.
  • Implemented secure authentication and user management with Clerk, and used MongoDB with Mongoose to store user data and chat history.
  • Added syntax highlighting (PrismJS), markdown rendering, and toast notifications to smooth out the experience.
Next.js Tailwind CSS MongoDB Mongoose Clerk OpenAI API
